Dungeon Index Site Plan

Justin Hansen

WDD 131

Overview

Purpose

this will be a website that can hold an index of monsters that can be used to build combat situations for Dungeons and Dragons combats. It will also allow the creation and hopefully the saving of Player Characters for future reference and for use as NPCs in future combats as well.

Audience

this is for the use of Dungeon Masters as well as players that are looking for references.

Dynamic elements

there will be the ability to sort through the different monsters based on different parameters (Challenge rating, alignment, type of creature, level for charaters)
